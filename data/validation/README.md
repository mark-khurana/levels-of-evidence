# Validation data

Reviewer validation of the automated extraction.

- `validation_samples_full.xlsx` — the complete reviewer validation workbook:
  - sheet `loe_accuracy`: stratified 5% recommendation sample (n = 1,179) with the
    extracted evidence level, the reviewer's confirmation, and the source recommendation;
  - sheet `extraction_completeness`: one randomly selected guideline per society (n = 77),
    automated vs reviewer recommendation counts.
- `validation_completeness.csv` — completeness counts per guideline (eTable 5).
- `validation_accuracy_by_society.csv` — evidence-level concordance by society (eTable 6).

Recommendation excerpts in the workbook are limited samples reproduced for the purpose
of methodological validation.
